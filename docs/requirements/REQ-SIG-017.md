## 1. 概述

### 1.1 信号描述
当前充电功率信号，用于显示充电过程中的实时功率。

### 1.2 相关需求
- REQ-IND-011: charge_light (充电灯控制)
- REQ-HYBRID-001: 充电状态显示

---

## 2. 信号规格

### 2.1 CAN 帧信息
| 字段 | 值 |
|------|-----|
| CAN ID | 0x302 |
| 来源 | CHG_POWER |
| 周期 | 100ms |

### 2.2 数据字段
| 字段 | 值 |
|------|-----|
| 名称 | charge_power |
| 字节位置 | [0, 1] (bytes 0-1) |
| 位宽 | 16 bits |
| 字节序 | little endian |
| 类型 | float |
| 公式 | x / 1000.0 |
| 单位 | kW |

### 2.3 有效范围
| 字段 | 值 |
|------|-----|
| 最小值 | 0 kW |
| 最大值 | 22 kW (慢充) / 100+ kW (快充) |

### 2.4 显示格式
- 格式: "{:.1f} kW"
- 精度: 1位小数
- 超时显示: "---"
